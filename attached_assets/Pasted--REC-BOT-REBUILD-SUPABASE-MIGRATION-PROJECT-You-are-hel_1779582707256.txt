# REC BOT REBUILD / SUPABASE MIGRATION PROJECT

You are helping rebuild and stabilize a large production Discord bot for a Madden Connected Franchise (CFM) league system called the REC League.

The current project is a migration from:
- an old Replit-based bot/database structure
TO:
- a new TypeScript Discord bot
- Supabase/Postgres backend
- selector-driven Discord UI system

The current codebase is partially migrated and contains:
- stale handlers
- dead references
- duplicated routing systems
- partial menu rewrites
- schema drift between code and database

Your job is to help COMPLETE and STABILIZE the rebuild, not create unrelated abstractions.

4. Finish `/new-server-setup`
